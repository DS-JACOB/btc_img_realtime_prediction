## btc_img_realtime_prediction

비트코인 가격 이미지 데이터를 활용한 비트코인 가격 예측
1. 학습 데이터 기간 선정
2. 특정 기간의 비트코인 가격을 이미지로 변환 
3. 해당 이미지와 유사한 이미지를 과거 비트코인 가격 이미지에서 검색
4. 유사한 이미지의 다음 이미지를 예측값으로 return
